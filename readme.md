## requirements

- Install [Git](https://git-scm.com/download/win)
- Install [Node](https://nodejs.org/en/download)

To check if properly installed
```
node -v
npm -v
git -v
```

```
git clone https://github.com/lawwman/countdown-app.git
```

## check IP
```
npm run check:ip
```

## receiving updates

```
git pull origin master
npm i
npm run dev
```
